## v0.2.0 - 2026-06-26

- 新增 PPT 演讲模式：支持 IP 演讲 PPT、直播分享课件、课程课件和全图化手绘 PPT 页面。
- 新增 PPT 工作流：导演规划卡 -> 样张确认 -> 分批生成 -> 整套 QA。
- 新增 PPT 页面类型库：封面页、大判断页、模块页、标准页、场景页、时间线页、方法页、收束页等。
- 新增 PPT 模式公开样例：`assets/examples/ppt-mode/`，包含 8 张 16:9 单页样例和一张 4x2 gallery 展示图。
- README 新增 PPT 模式说明和样例图展示。
- `SKILL.md` 和参考文档新增 PPT 模式触发、推荐、规划和 QA 规则。
- 保持向后兼容：原有角色资产、手绘插图、知识卡和执行 Agent 图解流程不变。

## v0.1.0 - 2026-06-22

- 同步内部 Skill 的 Agent 协作图解规则：知识卡默认采用“IP 主角 + 2-6 个执行 Agent”的表达。
- 新增推荐运行环境和环境分流：有图像生成工具时可直出图，没有时交付可复制 prompt 和返修 prompt。
- 强化第三张角色资产：动作 / 表情 / 小比例场景扩展图必须包含“主角 + Agent 协作”区。
- 增加方法论 / 步骤 / 流程类知识卡 QA：只有 IP 主角 + 文字、没有执行 Agent 或动作隐喻时判为图解感不足。
- README 补充“IP 主角 + 执行 Agent 的知识图解系统”定位。
- 首次整理公共发行候选版。
- 新增 `SKILL.md`、`README.md`、`LICENSE`、`.gitignore`、`assets/`、`references/`、`examples/` 和 `evals/`。
- README 增加多图 Gallery，展示工作流总览、角色资产、手绘插图模式和知识卡模式。
- 将具体平台绑定改写为通用“图像生成工具”表述。
- 明确用户角色资产、照片、主页截图和参考图的授权边界。
