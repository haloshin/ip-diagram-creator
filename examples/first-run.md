# 示例：第一次创建个人 IP 图解角色

## 用户输入

```text
我想做一个自己的极简手绘图解角色。我会发一张本人照片和我的主页截图，里面有昵称和简介。请先帮我提取信息，生成角色信息提取卡和角色固定资产 prompt，不要先问我要知识点。
```

## 期望处理

Agent 应先从用户授权材料中提取：

- 称呼。
- 领域。
- 简介。
- 气质。
- 外形锚点。
- 职业道具。
- 禁区。

然后输出：

1. 角色信息提取卡。
2. `01_角色主锚图` prompt。
3. `02_角色规范说明图` prompt。
4. `03_动作_表情_小比例场景扩展图` prompt。
5. 一个确认问题，等待用户确认角色资产。

角色未确认前，不进入内容图解。
