# PPT 演讲模式示例

这个示例用于触发 `ip-diagram-creator` 的 PPT 演讲模式。

## 适用场景

- 直播分享。
- 课程课件。
- 主题演讲。
- 案例复盘。
- 全图化手绘 PPT 页面。

## 示例输入

```text
我已经确认了个人 IP 角色。

现在我想做一套 12 页左右的直播分享 PPT，主题是：
《如何把 AI 学习变成一个真实可演示的作品》

受众是刚学了一段时间 AI、但还没有做出真实作品的人。

请先不要出图。请按 PPT 演讲模式，先给我整套导演规划卡：
- 每页是什么页面类型
- 视觉权重是 anchor / dense / breathing
- 每页承担什么沟通任务
- 是否放 IP 人物
- 信息密度大概多少
- 哪几页适合先做样张
```

## 期望输出

Agent 应先判断为 `PPT 演讲模式`，而不是普通知识卡或单张插图。

输出应包含：

- 整套目标。
- 默认尺寸：16:9。
- 初步页数。
- 风格方向。
- 每页导演规划卡。
- 1-2 页样张建议。
- QA 风险提示。

不应直接批量出图，也不应把每一页都做成高密度知识卡。
