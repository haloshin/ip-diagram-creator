Public README examples for `ip-diagram-creator` PPT mode.

These images demonstrate the PPT presentation mode as a page-type gallery: cover, big judgment, module page, standard page, scene page, timeline page, method page, and closing page.

These are public demo images authorized for this repository. Do not add user-private role assets, original photos, home-page screenshots, client materials, or non-public reference images to this folder.
